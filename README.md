# JoinMe

## Demployment

GitHub Actions configured to atomaticaly deploy after each commit. Demo url: https://join-me.azurewebsites.net

## Project setup

```bash
npm i
```

## Preparations

```bash
export MONGODB_CONNECT_URI=<MONGODB_CONNECT_URI>
export SESSION_SECRET=<SESSION_SECRET>
export PORT=<PORT>
```

## Build

```bash
npm run build
```

## Start

```bash
npm start
```
